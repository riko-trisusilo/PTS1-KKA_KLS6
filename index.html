Apa itu Koding?

Koding adalah cara kita memberikan perintah kepada komputer. Seperti ketika kita memberikan instruksi dalam permainan, koding adalah bahasa yang dimengerti komputer.

Mengapa Belajar Koding?

  • Melatih berpikir logis dan sistematis
  • Mengembangkan kreativitas
  • Memecahkan masalah dengan cara yang terstruktur
  • Mempersiapkan diri untuk masa depan digital

Konsep Dasar Koding:

  • Urutan (Sequence)
  • Perulangan (Loop)
  • Kondisi (If-Else)
  • Variabel (Variable)

Koding untuk Permainan Olahraga

Mari buat program sederhana tentang permainan olahraga menggunakan konsep koding dasar!

Contoh Kode: Menghitung Skor Permainan

// Program menghitung skor permainan
let skorTimA = 0;
let skorTimB = 0;

// Fungsi untuk menambah skor
function tambahSkor(tim) {
  if (tim === "A") {
    skorTimA = skorTimA + 1;
  } else if (tim === "B") {
    skorTimB = skorTimB + 1;
  }
  tampilkanSkor();
}

// Fungsi untuk menampilkan skor
function tampilkanSkor() {
  console.log("Skor Tim A: " + skorTimA);
  console.log("Skor Tim B: " + skorTimB);
}

Skor Tim A: 0

Skor Tim B: 0

Apa itu Kecerdasan Buatan (AI)?

Kecerdasan Buatan (AI) adalah kemampuan komputer untuk meniru kecerdasan manusia. AI dapat belajar, memecahkan masalah, dan membuat keputusan seperti manusia.

Contoh AI dalam Kehidupan:

  • Asisten virtual (Siri, Google Assistant)
  • Rekomendasi video di YouTube
  • Filter spam di email
  • Pengenalan wajah di ponsel

AI dalam Olahraga:

  • Analisis performa pemain
  • Strategi permainan
  • Wasit bantuan (VAR)
  • Prediksi hasil pertandingan

Bagaimana AI Bekerja dalam Permainan Ini?

AI dalam permainan ini menggunakan algoritma sederhana:

  • Pertama, mencari posisi untuk menang
  • Kedua, menghalangi pemain untuk menang
  • Ketiga, mengambil posisi strategis (tengah, sudut, atau sisi)
  • Terakhir, mengambil posisi yang tersedia

Aktivitas Kelas

1. Diskusi Kelompok: AI dalam Olahraga Favorit

Bentuk kelompok dan diskusikan bagaimana AI dapat digunakan dalam olahraga favorit kalian. Presentasikan hasil diskusi di depan kelas!

2. Membuat Algoritma Permainan

Buat algoritma sederhana untuk salah satu gerakan dalam permainan olahraga (contoh: cara melakukan servis dalam bulutangkis). Tuliskan langkah-langkahnya secara berurutan!

3. Proyek Koding Sederhana

Buat program sederhana menggunakan Scratch atau Blockly yang mensimulasikan salah satu aspek permainan olahraga (contoh: menghitung skor, menentukan pemenang, dll).
